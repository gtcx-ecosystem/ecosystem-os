# Opportunity

Problem definition and solution framing — the core case for why this product exists.

## Contents

| File                         | Description                                               |
| ---------------------------- | --------------------------------------------------------- |
| [`problem.md`](problem.md)   | Problem statement, market gap, and evidence of pain       |
| [`solution.md`](solution.md) | Proposed solution, value proposition, and differentiation |

## What belongs here

- Problem statements and evidence of the market gap
- Solution hypotheses and value proposition framing
- Why now — catalysts and timing rationale
- Core differentiation from alternatives

## What does NOT belong here

- Market sizing (→ `2-market/`)
- Customer research (→ `3-customers/`)
- Roadmap (→ `5-roadmap/`)
