# Fundraising Engine 30-60-90 Day Plan

## First 30 Days: Stand Up The Machine

Objective: make the fundraising office operational with manual workflows,
ClickUp, Gmail, Calendar, Drive, and agent support.

Deliverables:

1. ClickUp fundraising CRM configured.
2. `hello@gtcx.trade` labels, aliases, and calendar workflow configured.
3. Source database, query library, watchlist, and forensic backlog validated.
4. First 50 sources reviewed and first 25 backlog items loaded.
5. Daily brief running every weekday.
6. Outreach, LinkedIn, WhatsApp, call, and handoff templates in use.
7. Finance review and claims review gates active.
8. Drive folder structure and collateral inventory created.

Targets:

| Metric | Target |
| ------ | ------ |
| Source signals reviewed | 250+ |
| Qualified opportunities | 40+ |
| P1 opportunities | 15+ |
| Approved outreach sends | 20+ |
| Meetings booked | 8+ |
| Applications/proposals submitted | 3+ |
| Collateral assets drafted | 5+ |
| External sends without approval | 0 |

Decision at day 30:

- Confirm which lanes are producing the strongest signal: catalytic capital,
  sovereign/DFI pilots, investor capital, buyer revenue, accelerators/events,
  or partnerships.

## Days 31-60: Automate The Repeated Work

Objective: reduce manual drag and turn the source graph into a repeatable daily
intelligence system.

Deliverables:

1. Gmail-to-ClickUp n8n workflow.
2. Calendar-to-ClickUp n8n workflow.
3. Daily digest automation.
4. RSS and page-watch listener workflow for priority sources.
5. Weekly search-query sweep.
6. Dedupe and scoring rules refined from first-month data.
7. Country coverage expanded for priority markets.
8. Buyer/refiner/trader/vault watchlist expanded.

Targets:

| Metric | Target |
| ------ | ------ |
| Monitored source records | 150+ |
| Recurring search queries | 50+ |
| Qualified opportunities | 80+ cumulative |
| P1 opportunities | 25+ cumulative |
| Meetings booked | 20+ cumulative |
| Applications/proposals submitted | 8+ cumulative |
| Revenue/pilot conversations | 10+ cumulative |
| Automation-created tasks reviewed | 100% |

Decision at day 60:

- Decide whether ClickUp plus n8n is sufficient for another quarter or whether
  to start the Postgres-backed intelligence app immediately.

## Days 61-90: Build The Intelligence Layer

Objective: create the foundation for a GTCX-owned fundraising intelligence app
while keeping ClickUp as the execution layer.

Deliverables:

1. App schema finalized from the data model.
2. Source, organization, signal, and opportunity tables created.
3. Import pipeline for JSON source files.
4. ClickUp one-way sync prototype.
5. Dashboard for P1 opportunities, deadlines, stale tasks, and approvals.
6. Approval log prototype.
7. Country and commodity coverage roadmap.
8. Weekly leadership review pack.

Targets:

| Metric | Target |
| ------ | ------ |
| Monitored source records | 300+ |
| Recurring search queries | 75+ |
| Organizations in watchlist | 150+ |
| Qualified opportunities | 125+ cumulative |
| P1 opportunities | 40+ cumulative |
| Meetings booked | 35+ cumulative |
| Submitted applications/proposals | 12+ cumulative |
| Active strategic partnership/pilot conversations | 15+ |

Decision at day 90:

- Choose the next scale move: expand country coverage, build investor/contact
  graph, deepen commodity revenue pipeline, or accelerate app build.

## Weekly Cadence

| Day | Operating Rhythm |
| --- | ---------------- |
| Monday | Pipeline review, P1 prioritization, weekly source targets |
| Tuesday | Outreach block, warm-intro block, collateral production |
| Wednesday | Application/proposal block, finance review, claims review |
| Thursday | Partnership/pilot/revenue deal block, meeting prep |
| Friday | KPI report, lessons learned, source database cleanup |

## Governance

1. Founder approves strategic prioritization and all external commitments.
2. Chief of Staff Agent keeps the operating rhythm and clears blockers.
3. Finance Deal Economics Agent reviews all financial claims and terms.
4. Claims Compliance Reviewer Agent reviews all product, market, regulatory,
   ESG, traceability, and impact claims.
5. CRM ClickUp Agent ensures every active opportunity has status, owner, source,
   deadline, and next action.
