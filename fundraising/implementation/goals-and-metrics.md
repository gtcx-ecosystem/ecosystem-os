# Fundraising Engine Goals and Metrics

## North Star

Verified capital, revenue, pilot, and partnership opportunities advanced through
GTCX's Africa/Global South commodity trust infrastructure pipeline.

## Goals

| Goal | Target Outcome |
| ---- | -------------- |
| G1 Source coverage | Build the most comprehensive Africa/Global South fundraising and opportunity database for GTCX |
| G2 Pipeline velocity | Convert source signals into qualified opportunities with owners and next actions |
| G3 Strategic fit | Prioritize sovereign pilots, African banks/DFIs, commodity buyers, trade bodies, and Africa-focused capital |
| G4 Collateral readiness | Produce audience-specific decks, one-pagers, concept notes, and proposals quickly |
| G5 Financial rigor | Maintain clear budgets, runway, models, and deal economics for all serious opportunities |
| G6 Operating discipline | No active opportunity lacks owner, source URL, next action, and status |
| G7 Human control | No external send, submission, legal, financial, or relationship commitment happens without approval |

## Quality Gates

| Gate | Requirement |
| ---- | ----------- |
| Source gate | Every opportunity has source URL and source excerpt |
| Fit gate | Every opportunity has opportunity class, product match, and fit score |
| Owner gate | Every active opportunity has one accountable owner |
| Next-action gate | Every active opportunity has a concrete next action |
| Claims gate | Every external draft/collateral asset has claims review |
| Finance gate | Every term, valuation, price, budget, or revenue-share claim has finance review |
| Approval gate | Every external action has human approval recorded |
| Evidence gate | Every submitted/won item links to confirmation/evidence |
