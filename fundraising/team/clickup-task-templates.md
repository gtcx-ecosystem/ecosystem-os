# ClickUp Task Templates

Use these task templates to keep the agentic team consistent.

## Opportunity Task

Task name:

```text
{Organization} — {Opportunity Type} — {Country/Theme}
```

Description:

```text
Source URL:
Source channel:
Organization:
Opportunity class:
Funding / deal type:
Product match:
Why it matters:
Fit score:
Deadline:
Warm intro path:
Recommended next action:
Owner:
```

Checklist:

```text
- [ ] Source URL captured
- [ ] Organization/contact captured
- [ ] Product match assigned
- [ ] Fit score assigned
- [ ] Owner assigned
- [ ] Next action written
- [ ] Collateral needed identified
- [ ] Claims/finance review required? yes/no
```

## Outreach Draft Task

Task name:

```text
Draft outreach — {Organization} — {Audience}
```

Checklist:

```text
- [ ] Audience defined
- [ ] Desired action defined
- [ ] Draft prepared
- [ ] Relevant collateral linked
- [ ] Claims reviewed
- [ ] Finance reviewed if terms/pricing appear
- [ ] Human approval requested
- [ ] Sent or parked
```
