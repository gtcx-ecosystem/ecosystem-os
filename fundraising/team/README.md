# Fundraising Agentic Team

This package defines the agentic team that runs the GTCX fundraising engine
across sources, opportunities, outreach, applications, pilots, partnerships,
revenue deals, and investor operations.

## Operating Principle

The team should behave like a high-performing fundraising office, not a chatbot.
Agents gather intelligence, prepare work, maintain systems, draft materials, and
surface decisions. Humans approve relationship moves, external sends,
submissions, commitments, legal terms, financial terms, and strategic tradeoffs.
