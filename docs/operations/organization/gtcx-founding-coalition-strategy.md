# GTCX Founding Coalition: Continental Industry Initiative

## The Framing

This is not a pilot. This is the launch of continental commodity verification infrastructure, co-owned by African nations.

**The question isn't:** "Which country do we test in?"
**The question is:** "Which countries form the founding coalition?"

---

## Criteria for Founding Coalition Members

| Criteria                             | Why It Matters                                   |
| ------------------------------------ | ------------------------------------------------ |
| **Significant commodity production** | Must have skin in the game                       |
| **ASM presence**                     | Core use case for verification                   |
| **Regulatory pressure**              | FATF, EU, international scrutiny creates urgency |
| **Political will**                   | Leadership that wants to move                    |
| **Regional influence**               | Can bring neighbors along                        |
| **Existing relationships**           | Entry point exists                               |
| **Technical capacity**               | Can actually implement                           |
