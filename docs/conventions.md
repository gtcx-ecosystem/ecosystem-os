# Conventions

- **P35 v5:** unnumbered folders only under `docs/` (`gitbook/`, `overview/`, …).
- **Frontmatter:** `title`, `status`, `date`, `owner` on publish-facing pages.
- **Constitution / protocols:** link to `canon-os` — this repo holds fleet operator docs, not normative law.
- **GitBook mirrors:** `docs/reference/gitbooks/<product>/` — synced from owner repos; do not hand-edit.
- **GTM machine domain:** `ops/gtm/` (not under `docs/gtm/`).
