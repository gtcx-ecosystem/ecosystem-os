# Market Landscape

## Summary

Several companies claim pieces of the AI-native company operating system, but the full portfolio remains open.

Ecosystem OS should not position itself as another copilot, RAG app, workflow engine, or agent builder. The stronger category is a company operating system built around Kernel, Graph, Ledger, Surface, and operating modules for Ethos, Venture, Agency, Bridge, Agile, Fabric, and Assurance.

## Closest commercial analogues

| Company / product | Relevant claim | Closest Ecosystem OS layer |
| ----------------- | -------------- | -------------------------- |
| Microsoft Copilot / Agent 365 / Microsoft Graph | Enterprise agents grounded in Microsoft Graph and managed as agent assets. | Graph, Kernel, Bridge, Surface |
| Salesforce Agentforce | Autonomous agents across sales, service, marketing, commerce; Slack agentic OS direction. | Bridge, Agency, Surface |
| ServiceNow AI Control Tower / Workflow Data Fabric | Workflow automation, AI control tower, service operations, governance. | Fabric, Assurance, Bridge |
| Workday Illuminate / Agent System of Record | Governed agents across HR, finance, and planning; agent management as a system of record. | Ethos, Kernel, Assurance |
| Glean Work AI / Enterprise Graph | Enterprise search, assistant, agents, permission-aware enterprise graph. | Graph, RAG, Surface |
| Atlassian Rovo / System of Work | Teamwork graph, search, agents, project and collaboration context. | Agile, Graph, Bridge |
| Google Gemini Enterprise / Agentspace | Workplace AI platform with agent creation, data integrations, permissions, starter agents. | Bridge, Surface, Kernel |
| Workato Enterprise MCP | Enterprise MCP platform for governed agent access to tools and business systems. | Bridge, Fabric, integrations |

## Strategic gap

The market is converging toward four separate areas:

1. Enterprise graph and search.
2. Agent builders and orchestration.
3. Workflow/service operations.
4. Governance, risk, and provenance.

Ecosystem OS should combine these with the missing human and creative layers:

1. Ethos: culture, judgment, board, advisors, leadership, human profiles.
2. Venture: innovation, IP, product concepts, market theses.
3. Agency: creative, comms, marketing, brand expression.
4. Canon: publishing OS, separate from Ecosystem OS operations.
