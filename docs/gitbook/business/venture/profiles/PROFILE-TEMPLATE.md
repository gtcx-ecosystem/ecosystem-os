# Venture profile template

Copy for each build-lane repo under `profiles/<repo>.md`.

## Pilot client thesis

One paragraph: who this repo serves as an internal pilot client and the buyer promise.

## Fundraising hooks

- Investor narrative angle
- Evidence path (audit witness or pilot artifact)
- Capital rail or programme link

## GTM hooks

- ICP segment
- Agency campaign pack ref (if any)
- Demo / surface entry path

## Partnership hooks

- Integrator or sovereign partner type
- Handoff repo or coordination XR

## Evidence

| Witness | Path |
| ------- | ---- |
| Product goals | `<repo>/pm/spec/product-goals.json` |
| Latest audit | `<repo>/audit/evidence/` |
