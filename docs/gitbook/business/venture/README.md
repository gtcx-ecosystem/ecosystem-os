# Venture — innovation and IP operating system

Venture is the **portfolio and thesis** operating module for GTCX Ecosystem OS.

It owns strategic bets, product concepts, market theses, IP narratives, and the coordination layer between fleet repos (internal pilot clients) and external capital markets.

## What Venture owns (in ecosystem-os)

- Product portfolio and packaging narrative
- Strategic theses and innovation register
- Fleet venture profiles (each product repo as a venture profile)
- Fundraising engine backlog alignment
- Handoff to **venture-os** for product implementation

## What venture-os owns

The **Venture OS product** — fundraising, growth, visibility, investor relations, and founder workflows. See [`venture-os`](../../../venture-os/) at ecosystem root.

## Boundaries

- **Venture → Agency**: Venture invents (thesis, IP, strategy). Agency expresses (creative, comms, campaigns).
- **Venture → Canon**: Official publish surfaces; Venture maintains portfolio SoR until Canon promotion.
