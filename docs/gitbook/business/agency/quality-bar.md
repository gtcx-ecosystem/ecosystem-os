# Agency quality bar

This rubric defines what “world-class AI-native agency” means for outputs shipped through this module.

## Craft (baseline)

- **Typography**: deliberate hierarchy; consistent scale; readable density; no default-system mush
- **Grid & spacing**: consistent grid; spacing tokens; alignment discipline
- **Voice**: institutional, precise, non-hype; claims are specific and scoped
- **Exports**: clear target formats (web, PDF, slide, social); consistent naming and versioning

## World-class (studio bar)

- **Distinctiveness**: a recognizable signature (not generic AI aesthetic)
- **Coherence**: deck + one‑pager + social kit match the same system without drift
- **Credibility**: every material claim has a proof link (witness/source)
- **Speed without slop**: fast iteration with provenance and approvals intact

## Review checklist (ship)

- [ ] Brand rules applied (typography, grid, voice)
- [ ] Audience + intended action is explicit
- [ ] Claim→witness links complete
- [ ] License/attribution checked for any sourced assets
- [ ] Registry entry created/updated (ID, version, canonical link)
