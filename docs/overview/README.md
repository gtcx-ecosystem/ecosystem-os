# GTCX Ecosystem — vision and mission

## Mission

Replace **permission with proof** in African and global natural-resource markets — sovereign data, auditable transactions, AI-native operator experience.

Every commit either moves toward that mission or it does not.

## Vision

A multi-repo product fleet where:

- **Proof replaces permission** for market access
- **Sovereign data** stays under national and operator control
- **AI agents** operate under explicit constitutional and protocol law
- **Documentation** is crafted, audited, and refreshed like production code
