# Nillion Private Data Browser Extension

## Purpose

Evaluate a privacy‑preserving browser extension for sensitive data access and verification workflows.

## Use Cases

- Secure viewing of compliance evidence
- Private review of verification proofs
- Redacted data access for auditors

## Risks

- Dependency on third‑party extension lifecycle
- Browser compatibility and user adoption

## Next Steps

- Prototype minimal extension
- Assess threat model and auditability
- Decide on build vs integrate
