# FFI RCO Integration Framework Launch

## Purpose

Define the integration framework between GTCX and external regulatory or compliance organizations (RCOs).

## Scope

- Data exchange requirements
- Evidence formats and audit trails
- Rule pack alignment

## Integration Model

- Read‑only data sharing where possible
- Proofs exchanged, not raw data
- Signed audit exports

## Deliverables

- Integration checklist
- Sample data contracts
- Pilot launch plan
