# Biometric Security: Fingerprint Implementation

## Purpose

Define biometric options for secure identity confirmation without compromising cost or usability.

## Options

- **Device biometric**: use phone fingerprint for TradePass confirmation
- **Biometric cards**: higher security but higher cost

## Recommendation

- Use device biometrics for pilots
- Reserve biometric cards for high‑risk roles

## Risks

- Cost constraints for large‑scale rollout
- User onboarding and device compatibility
