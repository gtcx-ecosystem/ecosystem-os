3. **Coordinate** — switch to owner repo; run gates (Bridge)
4. **Ship** — implement story; micro-commit (Agile)
5. **Verify** — ops:check + pillar witnesses (Assurance)
6. **Learn** — auditNotes → backlog intake (Graph)
